Docket使用

基本命令

查看镜像

命令:docker images

查找镜像

命令：docker serach

下载镜像

命令:docker pull

删除镜像

命令：docker rmi -f 容器id

容器命令

容器启动

docker run [可选参数] image

查询所有运行容器

docker ps

退出容器

exit #容器停止并退出

ctrl p q #容器不停止退出

容器删除

docker stop 容器id #停止当前正在运行的容器

docker kill 容器id #强制停止当前容器

docker rm 容器id #删除容器

小技巧（删除所有容器）：docker rm $(docker ps -a -q)

后台启动容器

docker start 容器id # 删除所有容器

docker restart 容器id #重启容器

查看日志

docker logs -f -t –tail 容器

容器数据卷

docker原理是将应用和环境打包成一个镜像

至于数据是存储在本地或者是其他地方，意思是数据共享。即将容器中的数据目录挂在到本地或者其他位置

使用数据卷

docker run -it -v 主机目录：本机目录

Dockerfile

Dockerfile就是用来构建镜像

构建步骤：

  • 编写dockerfile文件
  • docker build 构建一个镜像
  • docker run 运行镜像
  • docker push
